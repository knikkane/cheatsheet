# cheatsheet
simple bash function to quickly access command cheatsheets from .md file

## Usage
```
cheat                 # Display a specific topic
cheat --list          # List all available topics
```

## Cheat Sheet Format
`cheatsheet.md` following structure:
```
## Topic Name
Content for this topic

## Another topic
More content here
```

### Example Structure:
```
## git
Useful Git commands …

## Tmux
Tmux shortcuts …
```
